Biến CSS

Khai báo một lần ở :root, dùng lại bằng var(--ten). Đổi màu chủ đạo cả trang chỉ cần sửa một dòng.

Thẻ dùng biến cục bộ

Padding của thẻ này lấy từ biến --card-pad khai báo ngay trong .card.

Thử bật Dark mode trong cài đặt hệ điều hành (hoặc DevTools → Rendering → emulate prefers-color-scheme) — toàn trang tự đổi sáng/tối nhờ biến CSS.
