Ba cách chèn CSS

External màu nền lấy từ css/main.css.

Internal màu nền lấy từ thẻ <style>.

Inline màu nền đặt ngay trên thẻ bằng thuộc tính style.

Cascade — ai thắng khi xung đột?

Class này khai báo green ở internal, nhưng inline red thắng.

Đoạn này dùng !important → luôn xanh.

Kế thừa (inheritance)

Cả trang dùng font Arial chỉ vì khai báo font-family một lần ở body trong main.css — nhóm thuộc tính văn bản tự truyền xuống con.

Best practice: dự án thật gần như luôn dùng External CSS. Mở DevTools → tab Styles để thấy quy tắc nào bị gạch ngang (bị ghi đè).
