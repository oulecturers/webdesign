3) Bố cục nhiều cột (multicolumn)

HTML là ngôn ngữ đánh dấu chuẩn để tạo trang web. CSS quy định cách trình bày. Bố cục nhiều cột chia nội dung dài thành các cột như báo/tạp chí, giúp người đọc đỡ mỏi mắt khi dòng quá dài. Trình duyệt tự cân nội dung sang các cột. Bạn có thể điều chỉnh số cột bằng column-count, khoảng cách bằng column-gap và thêm đường kẻ giữa các cột bằng column-rule. Đây là cách nhanh để dàn văn bản nhiều cột mà không cần Flexbox hay Grid.
