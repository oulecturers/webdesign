2) box-sizing — TRỌNG TÂM THI

Cùng khai báo width:240px; padding:30px; border:3px:

content-box (mặc định) → rộng THỰC ≈ 306px
border-box → rộng THỰC = 240px (padding+border tính GỘP vào)

Best practice: đặt * { box-sizing: border-box; } đầu file CSS để "con số khai báo = kích thước nhìn thấy".
