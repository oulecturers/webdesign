8 phong cách thiết kế web phổ biến

Mỗi phong cách gắn với một vài kỹ thuật CSS cốt lõi. Mục tiêu của bài là nhận diện phong cách qua dấu hiệu thị giác, không phải học thuộc.

2.5 Design Styles Nhận diện Có demo tương tác

Rê chuột / bấm vào các nút trong từng thẻ để cảm nhận. Một sản phẩm thật thường pha trộn nhiều phong cách — quan trọng là biết gọi tên kỹ thuật tạo ra chúng.

1 · Minimalism (Tối giản)

Cốt lõi: nhiều khoảng trắng, typography lớn, rất ít màu.

Less is more

portfolio · blog

font-weight: 300; letter-spacing: 1px; /* nhiều whitespace, 1–2 màu */

Nhận diện: trang đề cao nội dung, gọn gàng, gần như đơn sắc.

2 · Flat Design (Phẳng)

Cốt lõi: màu phẳng tươi, bo góc nhẹ, bóng mờ tối giản.

Hover tôi ↑
background: #FFDD67; border-radius: 10px; box-shadow: 0 4px 6px rgba(0,0,0,.1);

Nhận diện: không hiệu ứng 3D, hình khối & icon phẳng, màu rực.

3 · Skeuomorphism (Mô phỏng thực)

Cốt lõi: gradient + nhiều lớp box-shadow tạo khối 3D thật.

background: linear-gradient(#fefefe,#cfcfcf); box-shadow: 0 6px 0 #888, 0 8px 10px rgba(0,0,0,.4);

Nhận diện: nút/đối tượng trông như vật thật, có chiều sâu, texture.

4 · Dark Mode (Chế độ tối)

Cốt lõi: biến CSS + prefers-color-scheme (đây có nút bật/tắt thật).

Nội dung mẫu
:root { --bg:#fff; --fg:#111; } .dark { --bg:#111; --fg:#f0f0f0; } /* @media (prefers-color-scheme: dark) */

Nhận diện: nền tối, chữ sáng; thường có công tắc sáng/tối.

5 · Brutalism (Thô, phá cách)

Cốt lõi: viền đen dày, bóng cứng (không nhoè), chữ IN HOA.

border: 3px solid #000; box-shadow: 6px 6px 0 #000; /* bóng đặc, sắc */ text-transform: uppercase;

Nhận diện: tương phản mạnh, "xấu có chủ đích", thô mộc.

6 · Glassmorphism (Kính mờ)

Cốt lõi: rgba() + backdrop-filter: blur() + viền sáng mảnh.

Mảnh kính mờ
backdrop-filter
background: rgba(255,255,255,.15); backdrop-filter: blur(10px); border: 1px solid rgba(255,255,255,.35);

Nhận diện: panel trong suốt mờ, thấy nền nhoè phía sau.

7 · Neumorphism (Tân ảnh khối)

Cốt lõi: nền cùng màu khối + box-shadow kép sáng/tối. Bấm để lõm.

box-shadow: 6px 6px 12px #bebebe, -6px -6px 12px #ffffff; /* :active → inset = lõm xuống */

Nhận diện: khối nổi mềm cùng tông nền, như nặn từ đất sét.

8 · Material Design (Google)

Cốt lõi: "độ cao" (elevation) bằng bóng nhiều lớp, màu đậm, chuyển động.

box-shadow: 0 3px 6px rgba(0,0,0,.16), 0 3px 6px rgba(0,0,0,.23); /* hover → bóng cao hơn (elevation) */

Nhận diện: thẻ nổi có bóng tinh tế, nút nâng lên khi tương tác.

Tự kiểm tra. App ngân hàng số hiện đại, nền có ảnh, panel trong suốt → Glassmorphism. Portfolio đề cao nội dung, ít màu, nhiều khoảng trắng → Minimalism. Nút nổi mềm cùng tông nền → Neumorphism.
