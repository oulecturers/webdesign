Bộ icon Font Awesome

Hàng nghìn icon vector chỉ với một dòng CDN và một thẻ <i>. Đổi cỡ, màu, xoay/lật, làm động, xếp chồng — tất cả bằng class CSS.

Cần Internet. Trang này tải icon từ CDN Font Awesome. Nếu mở offline (hoặc CDN bị chặn), icon sẽ không hiện — phần khung bài học vẫn đầy đủ. Có thể tự host (mục 10).

1 Font Awesome & cách nhúng Cơ bản

Font Awesome là thư viện icon dạng icon font / SVG phổ biến nhất. Chỉ cần thêm 1 thẻ <link> CDN vào <head> là dùng được mọi icon.

<!-- Đặt trong <head> -->
<link rel="stylesheet"
      href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.2/css/all.min.css">

Sau khi nhúng, mỗi icon là một thẻ <i> (hoặc <span>) mang class của icon đó. Tìm tên icon tại trang chính thức fontawesome.com/icons.

Bản Free đủ cho học tập (hàng nghìn icon). Một số icon & nhóm fa-regular nét mảnh thuộc gói Pro.

2 Cú pháp icon & 3 nhóm style Cơ bản

Mỗi icon cần 2 class: class nhóm style + class tên icon.

<i class="fa-solid fa-house"></i>     /* đặc (mặc định) */
<i class="fa-regular fa-heart"></i>   /* nét rỗng */
<i class="fa-brands fa-github"></i>   /* thương hiệu */

3 Kích thước icon Cơ bản

Icon kế thừa font-size của chữ xung quanh. Phóng to nhanh bằng class fa-xs → fa-2xl, hoặc bội số fa-2x … fa-10x.

Kết quả · fa-xs → fa-5x
xs 1x lg 2x 3x 5x 
<i class="fa-solid fa-camera fa-3x"></i>
/* hoặc tự đặt cỡ bằng CSS: */
.icon { font-size: 48px; }

4 Màu & tạo kiểu Cơ bản

Icon ăn theo color như chữ thường — đổi màu, đổ bóng… đều qua CSS.

Kết quả
.heart  { color: #E63946; }
.shadow { color: #6c5ce7; text-shadow: 2px 2px 4px rgba(108,92,231,.4); }

fa-fw ép mọi icon cùng bề rộng để thẳng hàng. fa-ul/fa-li thay chấm đầu dòng bằng icon.

6 Hiệu ứng động Nâng cao

Thêm 1 class là icon tự chuyển động — rất hợp làm loading, chuông thông báo, nút yêu thích.

Kết quả · các icon đang chuyển động
<i class="fa-solid fa-spinner fa-spin"></i>   /* xoay tròn */
<i class="fa-solid fa-heart fa-beat"></i>     /* đập nhịp */

8 Xếp chồng icon (fa-stack) Nâng cao

Đặt một icon nhỏ lên trên một icon lớn — ví dụ icon trong vòng tròn, dấu cấm.

Kết quả
<span class="fa-stack">
  <i class="fa-solid fa-circle fa-stack-2x"></i>  /* nền lớn */
  <i class="fa-solid fa-check fa-stack-1x"></i>  /* icon nhỏ phía trên */
</span>

10 Tra cứu & mẹo Tổng kết

Bảng tra nhanh class hay dùng
ClassTác dụng
fa-solid / fa-regular / fa-brandsChọn nhóm style của icon.
fa-2x … fa-10x, fa-lg, fa-xsPhóng to/thu nhỏ icon.
fa-fwBề rộng cố định — icon thẳng cột.
fa-spin, fa-beat, fa-bounce, fa-shake, fa-fadeHiệu ứng động.
fa-rotate-90/180/270, fa-flip-horizontal/verticalXoay & lật.
fa-stack + fa-stack-1x/fa-stack-2xXếp chồng icon.

Mẹo. Icon chỉ để trang trí nên thêm aria-hidden="true"; icon mang nghĩa thì kèm chữ hoặc title/aria-label cho người dùng trình đọc màn hình.

Tự host (không cần CDN). Tải gói Font Awesome về, đặt vào thư mục dự án rồi <link> tới css/all.min.css cục bộ — chạy được cả khi offline.

Lưu ý. Icon không hiện thường do: sai tên icon, thiếu class nhóm (fa-solid…), chưa nhúng CDN, hoặc icon đó thuộc gói Pro.
