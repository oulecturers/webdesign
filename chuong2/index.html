Chương 2 — Xây dựng giao diện Web với HTML & CSS

Bộ code mẫu theo từng mục · mở bất kỳ file .html nào bằng trình duyệt là chạy được

Cách dùng: mỗi mục là một thư mục, mỗi bài là một file độc lập, có chú thích tiếng Việt ngay trong mã. Đọc mã → mở bằng trình duyệt → so sánh kết quả. Khuyến nghị mở bằng VS Code + Live Server để tự nạp lại khi sửa.
