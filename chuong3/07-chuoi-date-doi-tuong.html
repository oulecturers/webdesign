1 Chuỗi: truy cập cơ bản Cơ bản

let s = "Javascript";      // đặt trong "..." hoặc '...'

s.length;       // 10  — độ dài (thuộc tính, KHÔNG có ngoặc)
s.charAt(0);    // "J" — ký tự tại vị trí 0
s[0];           // "J" — cách viết gọn hơn

// chuỗi là BẤT BIẾN (immutable): mọi phương thức đều trả CHUỖI MỚI
s.toUpperCase(); // "JAVASCRIPT" — nhưng s vẫn là "Javascript"

2 Hoa/thường · tìm kiếm Cơ bản

let s = "Javascript is simple, Javascript is great!";

s.toUpperCase();          // "JAVASCRIPT IS ..."
s.toLowerCase();          // "javascript is ..."
s.indexOf("Javascript");  // 0   — vị trí đầu tiên
s.lastIndexOf("Javascript"); // 22 — vị trí cuối cùng
s.includes("simple");     // true
s.startsWith("Java");     // true
s.trim();                 // cắt khoảng trắng 2 đầu — luôn dùng với input người dùng!

So sánh không phân biệt hoa thường: đưa cả hai về cùng cỡ chữ trước — a.toLowerCase() === b.toLowerCase(). Tìm kiếm trong danh sách món cũng vậy.

4 Số ↔ chuỗi Cơ bản

// CHUỖI → SỐ
parseInt("123.56");   // 123    — lấy phần nguyên, đọc tới khi hết số
parseFloat("123.56"); // 123.56
Number("123.56");     // 123.56 — nghiêm khắc: cả chuỗi phải là số
parseInt("abc");      // NaN    — câu trắc nghiệm số 1!

// SỐ → CHUỖI
(1/3).toFixed(3);     // "0.333" — làm tròn 3 chữ số thập phân (trả CHUỖI)
(75).toString(2);     // "1001011" — đổi sang NHỊ PHÂN (cơ số 2)
(255).toString(16);   // "ff" — hệ 16 (quen thuộc với mã màu CSS!)

Phân biệt cho kỳ thi: toFixed() là phương thức của số (trả chuỗi làm tròn) — nó không chuyển chuỗi thành số. Nhóm chuyển chuỗi → số là Number(), parseInt(), parseFloat().

5 Template literal ES6

Chuỗi trong dấu backtick `...`: chèn biến bằng ${...}, xuống dòng thoải mái — thay hẳn kiểu nối chuỗi bằng +.

// cũ: nối bằng + — dễ sai, khó đọc
let li = "<li>" + ten + " - " + gia + "đ</li>";

// mới: template literal — nhìn là hiểu
let li = `<li>${ten} - ${gia}đ</li>`;

for (let i = 1; i <= 3; i++)
  console.log(`${i} * 2 = ${2 * i}`);   // 1 * 2 = 2 …  (chèn được cả BIỂU THỨC)
Thử ngay — điền tên vào mẫu câu
// `Chào ${ten}, món ${mon} của bạn: ${gia}đ`

6 Date: tạo & bẫy "tháng từ 0" Cơ bản

new Date();                        // thời điểm HIỆN TẠI
new Date("2017-05-27");            // từ chuỗi ISO (yyyy-mm-dd)
new Date(2017, 4, 27, 10, 30, 0);  // năm, THÁNG(0-11), ngày, giờ, phút, giây
                                    //           ^^^ 4 = THÁNG 5 !!!

Bẫy kinh điển: trong Date, tháng đếm từ 0 (0 = tháng 1 … 11 = tháng 12). new Date(2025, 0, 1) là 1/1/2025. Riêng ngày thì vẫn đếm từ 1. Rất nhiều bug ngày tháng sinh ra từ đây.

4 định dạng chuỗi ngày mà new Date("...") hiểu: ISO "2017-05-27" (khuyên dùng) · Short "05/27/2017" (kiểu Mỹ: tháng trước ngày sau!) · Long "May 27 2017" · Full "Saturday May 27 2017".

8 Object: key/value · this Cơ bản

Object gom các cặp key: value (thuộc tính) và có thể chứa phương thức (hàm). Cú pháp literal { } là cách thường dùng.

let student = {
  firstName: "Le",
  lastName:  "Duong",
  gender:    "Male",
  fullName:  function () {                    // PHƯƠNG THỨC
    return this.firstName + " " + this.lastName;
  }                                             // this = object đang gọi phương thức
};

student.firstName;      // "Le"  — chấm (thường dùng)
student["firstName"];   // "Le"  — ngoặc vuông (khi key nằm trong biến)
student.fullName();     // "Le Duong"
student.email = "d@ou.edu.vn";  // thêm thuộc tính mới lúc nào cũng được
Thử ngay — this trỏ về ai?
// this.firstName nghĩa là "firstName của object NÀY"

this là gì? Trong phương thức, this trỏ tới chính object đang gọi. this sẽ gặp lại (và gây bẫy với arrow function) khi xử lý sự kiện DOM ở bài 3.7–3.8.

9 Destructuring object · chuyện về with Nâng cao

// trích thuộc tính vào biến cùng tên — một dòng lấy nhiều giá trị
const { firstName, lastName } = student;
console.log(firstName);              // "Le"

// đổi tên biến + giá trị mặc định
const { firstName: ten, tuoi = 18 } = student;

// gộp 2 object bằng spread (bài 3.6.1)
const full = { ...student, lop: "WD01" };

with — chỉ để biết, tuyệt đối không dùng: giáo trình có giới thiệu with (obj) {…} để viết gọn, nhưng nó đã lỗi thời, bị cấm trong strict mode, khó đọc và khó tối ưu. Thay bằng destructuring như trên.

10 Mẹo & lỗi thường gặp Tổng kết

Lỗi 1. getMonth() trả 0–11 — hiển thị tháng phải +1; tạo new Date(2025, 5, 1) là tháng 6 chứ không phải tháng 5.

Lỗi 2. Tưởng phương thức chuỗi sửa chuỗi gốc — chuỗi bất biến, phải gán lại: s = s.replace("js", "JavaScript"). Muốn thay tất cả: replaceAll (hoặc vòng lặp while (s.indexOf(...) >= 0) như đề trắc nghiệm câu 13).

Lỗi 3. Quên this. trong phương thức: return firstName → ReferenceError; phải là this.firstName.

Mẹo. Nối chuỗi có biến → luôn template literal. Ngày tháng hiển thị cho người Việt → toLocaleDateString("vi-VN"). Lấy nhiều thuộc tính → destructuring một dòng.
