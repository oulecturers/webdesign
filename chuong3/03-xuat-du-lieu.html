1 Console & DevTools Cơ bản

Console là công cụ chính để debug và kiểm tra dữ liệu.

  • Mở bằng F12 (Windows) / ⌘⌥I (macOS) → tab Console.
  • Mọi lỗi JavaScript của trang đều hiện ở đây, kèm tên file và số dòng — bấm vào là nhảy tới chỗ lỗi.
  • Có thể gõ lệnh JS trực tiếp vào Console để thử nhanh (REPL).

Thói quen vàng: khi trang "không chạy", việc đầu tiên là mở Console. 90% trường hợp câu trả lời nằm sẵn ở dòng chữ đỏ.

2 log · info · warn · error Cơ bản

console.log("Thông tin thường (phổ biến nhất khi debug)");
console.info("Thông tin hữu ích");
console.warn("Cảnh báo — chữ vàng");
console.error("Lỗi — chữ đỏ");

// log nhiều giá trị một lúc, cách nhau dấu phẩy:
console.log("Điểm:", 8.5, { ten: "An" });

3 console.table · console.dir Nâng cao

Xem mảng đối tượng (như danh sách món của "Góc Cà Phê") dạng bảng — dễ đọc gấp nhiều lần console.log.

const menu = [
  { ten: "Cà phê sữa", gia: 25000 },
  { ten: "Bạc xỉu",    gia: 29000 },
  { ten: "Trà đào",    gia: 35000 },
];
console.table(menu);   // hiển thị thành bảng có cột index / ten / gia
console.dir(document.body);  // xem object dạng cây (khác log với phần tử DOM)
Thử ngay — console.table(menu)
// bấm để xem bảng (bản thật nằm trong Console F12)

4 time · group · assert · clear Nâng cao

Phương thứcCông dụng
console.time(nhãn) / timeEnd(nhãn)Đo thời gian chạy một đoạn code
console.group(nhãn) / groupEnd()Gom các log thành nhóm gấp/mở được
console.assert(điều_kiện, msg)Chỉ in msg khi điều kiện sai
console.clear()Xoá sạch console
 
console.time("Loop");
for (let i = 0; i < 1_000_000; i++) {}   // 1_000_000: dấu _ chỉ để dễ đọc
console.timeEnd("Loop");                  // Loop: 3.2 ms (tuỳ máy)

console.assert(diem >= 0, "Điểm không được âm!");

5 alert · confirm · prompt Cơ bản

Ba hộp thoại có sẵn của trình duyệt — tiện cho demo, nhưng chặn toàn bộ trang cho đến khi người dùng bấm nút.

HàmHiển thịTrả về
alert(msg)Thông báo + nút OKundefined
confirm(msg)Câu hỏi + OK / Canceltrue (OK) hoặc false (Cancel)
prompt(msg)Ô nhập + OK / CancelChuỗi đã nhập, hoặc null nếu Cancel
 
let name = prompt("Tên của bạn:");
if (confirm("Bạn chắc chắn?")) {
  console.info("Hello " + name);
}
Thử ngay — chuỗi prompt → confirm → kết quả
// bấm để trải nghiệm 3 hộp thoại nối nhau

Lưu ý thực tế: alert/confirm/prompt chặn giao diện và trông thô — chỉ dùng để học/demo nhanh. Web thật hiển thị thông báo bằng thành phần HTML/CSS (modal, toast) đẹp và không chặn trang. Nhớ: prompt luôn trả về chuỗi — muốn tính toán phải đổi kiểu.

6 Ghi ra trang (DOM) Cơ bản

Cách web thật hiển thị dữ liệu: gán vào textContent / innerHTML của một phần tử (học sâu ở bài 3.8).

document.getElementById("kq").textContent = "Chỉ chữ — an toàn";
document.getElementById("kq").innerHTML  = "Có <b>thẻ HTML</b> — cẩn thận XSS";

// document.write("..."): CHỈ thấy trong sách cũ — gọi sau khi trang

7 Mẹo & lỗi thường gặp Tổng kết

Mẹo 1. Dữ liệu dạng mảng object → console.table(); object lồng sâu → console.dir(); nghi ngờ chậm → console.time().

Lỗi 1. prompt trả về chuỗi: prompt("Tuổi?") + 1 với input 20 cho "201" chứ không phải 21. Bọc Number() trước khi cộng.

Lỗi 2. Người dùng bấm Cancel → prompt trả về null — phải kiểm tra trước khi dùng.

Ghi nhớ. Debug → Console. Demo nhanh → dialog. Web thật → ghi ra DOM (textContent an toàn hơn innerHTML).
