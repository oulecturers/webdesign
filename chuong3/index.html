Chương 3 — Lập trình JavaScript căn bản

Bộ code mẫu theo từng mục · mở bất kỳ file .html nào bằng trình duyệt là chạy được · mỗi bài đều có demo tương tác bấm-thử-ngay

Cách dùng: mỗi bài là một file độc lập, có chú thích tiếng Việt ngay trong mã. Đọc mã → mở bằng trình duyệt → bấm thử các demo → mở DevTools (F12) tab Console để xem thêm. Khuyến nghị mở bằng VS Code + Live Server (bài 13 — fetch — cần chạy qua Live Server hoặc có mạng).

Phần 1 · Ngôn ngữ JavaScript (3.1 – 3.6)

3.1

Tổng quan JavaScript

JS là gì, làm được gì, cách nhúng, defer/async, chương trình đầu tiên.

 3.2

Biến · Kiểu dữ liệu · Toán tử

let/const/var, primitive vs reference, == vs ===, truthy/falsy.

 3.3

Xuất dữ liệu

console.log/table/time…, alert/confirm/prompt, ghi ra trang.

 3.4

Cấu trúc điều khiển

if/else, ba ngôi, switch, for/while, for…of, break/continue.

 3.5

Hàm

Khai báo, tham số mặc định, rest, function expression, arrow.

 3.6.1

Mảng

push/pop/splice, map/filter/reduce, sort, destructuring, spread.

 3.6.2-4

Chuỗi · Date · Object

Phương thức chuỗi, template literal, Date (tháng từ 0!), this.

Phần 2 · Tương tác trang web (3.7 – 3.9)

3.7

Xử lý sự kiện

addEventListener, sự kiện chuột/bàn phím/form, preventDefault.

 3.8

HTML DOM

querySelector, textContent vs innerHTML (XSS), classList, tạo/xoá nút, delegation, setInterval.

 3.9

HTML BOM · Lưu trữ

location, history, navigator, cookie, localStorage/sessionStorage.

Phần 3 · Đồ hoạ, thư viện & web hiện đại (3.10 – 3.12)

3.10

Canvas & SVG

Vẽ hình bằng JS, bóng nảy, SVG tương tác, chọn Canvas hay SVG.

 3.11

jQuery

Đọc-hiểu code cũ: selector, hiệu ứng fade/slide, đối chiếu vanilla JS.

 3.12

JSON · fetch · async/await

Bất đồng bộ, gọi API, try/catch, render dữ liệu ra DOM, CORS.
