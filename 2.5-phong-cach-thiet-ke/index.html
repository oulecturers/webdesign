8 phong cách thiết kế web phổ biến

Nhận diện phong cách + kỹ thuật CSS cốt lõi · ← về mục lục

1. Minimalism (Tối giản)

Cốt lõi: nhiều khoảng trắng, typography lớn, ít màu.

Less is more

portfolio · blog

2. Flat Design (Phẳng)

Cốt lõi: màu phẳng tươi, border-radius nhẹ, box-shadow mờ.

Hover tôi ↑

3. Scrollytelling (Kể chuyện bằng cuộn)

Cốt lõi: section full-height (100vh), scroll-behavior:smooth, hiệu ứng theo cuộn.

Khám phá vũ trụ → cuộn để dẫn chuyện

4. Skeuomorphism (Mô phỏng thực tế)

Cốt lõi: box-shadow/gradient nhiều lớp tạo khối 3D, texture.

5. Dark Mode (Chế độ tối)

Cốt lõi: biến CSS + prefers-color-scheme (ở đây có nút bật/tắt thật).

Nội dung mẫu

6. Brutalism (Thô, phá cách)

Cốt lõi: border dày đen, box-shadow cứng (không nhoè), uppercase.

7. Glassmorphism (Kính mờ)

Cốt lõi: rgba() + backdrop-filter:blur() + viền sáng.

Mảnh kính mờ
backdrop-filter

8. Neumorphism (Tân ảnh khối)

Cốt lõi: nền cùng màu khối + box-shadow kép (sáng + tối). Bấm để lõm.

Tự kiểm tra: App ngân hàng số, hiện đại, nền có ảnh → Glassmorphism. Portfolio đề cao nội dung, ít màu → Minimalism.
