Các thẻ đa phương tiện

1) iframe — nhúng một trang khác (chạy offline)

2) iframe — nhúng video YouTube (cần internet)

3) video — thẻ chuẩn HTML5

Bỏ file media/sample.mp4 vào cạnh file này để phát thật. Thuộc tính: controls, autoplay, muted, loop.

4) audio — thẻ chuẩn HTML5

Bỏ file media/sample.mp3 để nghe thật.

5) canvas — vẽ bằng JavaScript (raster)

Trình duyệt không hỗ trợ canvas.

6) svg — đồ hoạ vector (không vỡ khi phóng to)

Canvas vs SVG: canvas vẽ pixel qua JS (phóng to bị vỡ), hợp game/ảnh động; SVG là vector (sắc nét mọi kích thước), hợp logo/biểu đồ/icon.
